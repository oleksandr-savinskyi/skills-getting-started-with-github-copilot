# OctoAcme Project Management Docs

This README provides a summary of the project management processes used by OctoAcme and direct links to detailed documentation.

## Project Management Process Summary

OctoAcme employs an iterative, customer-first approach to project delivery with clear roles and lifecycle stages:

- **Principles:** Customer value, iterative delivery, clear ownership, data-informed decisions, psychological safety
- **Lifecycle:** Initiation → Planning → Execution → Release → Close/Retrospective
- **Roles:** Project Manager (PM), Product Manager (PdM), Developers, QA, Stakeholders
- **Artifacts:** Project Charter, Roadmap, Backlog, Acceptance Criteria, Risk Register, Retrospectives
- **Communication:** Weekly PM/PdM sync, delivery team standups, stakeholder updates
